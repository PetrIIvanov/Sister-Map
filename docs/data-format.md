# Формат данных data.json

Файл `data.json` — массив объектов. Каждый объект описывает одну страну.

## Поля

| Поле            | Тип    | Описание                                      | Пример            |
|-----------------|--------|-----------------------------------------------|-------------------|
| `code`          | string | ISO alpha-2 код страны                        | `"RU"`            |
| `country_name`  | string | Название страны на русском                    | `"Россия"`        |
| `word`          | string | Слово «сестра» на местном языке               | `"сестра"`        |
| `transcription` | string | Фонетическая транскрипция (IPA)               | `"[sʲɪˈstra]"`    |
| `ru_reading`    | string | Произношение, записанное кириллицей           | `"сестра"`        |
| `group`         | string | Языковая группа                               | `"slavic"`        |
| `info`          | string | Историко-лингвистическая справка              | `"Классический…"` |

## Языковые группы

- `slavic` — славянская
- `germanic` — германская
- `romance` — романская
- `uralic` — уральская
- `baltic` — балтийская
- `other` — прочие (греческий, албанский, ирландский, турецкий)
