# Цветовая схема языковых групп

Используется для раскраски маркеров на карте (CircleMarker).

| Группа     | Название       | Цвет     | HEX       | RGB             |
|------------|----------------|----------|-----------|-----------------|
| slavic     | Славянская     | Красный  | `#e74c3c` | `(231,76,60)`   |
| germanic   | Германская     | Синий    | `#3498db` | `(52,152,219)`  |
| romance    | Романская      | Зелёный  | `#2ecc71` | `(46,204,113)`  |
| uralic     | Уральская      | Фиолетовый| `#9b59b6` | `(155,89,182)`  |
| baltic     | Балтийская     | Оранжевый| `#e67e22` | `(230,126,34)`  |
| other      | Прочие         | Серый    | `#95a5a6` | `(149,165,166)` |

## Принцип подбора
- Цвета достаточно контрастные, чтобы группы различались на мелком масштабе
- Используются стандартные цвета Folium (передаются строкой: `"red"`, `"blue"` и т.д.)
- HEX и RGB — для справки, при необходимости кастомной стилизации
