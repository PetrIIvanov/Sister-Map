# Архитектура SisterMap

## Схема

```
data.json ──► map.py ──► sister_map.html
```

## Компоненты

### 1. data.json (вход)
Исходные данные: 33 записи с полем `code` (ISO alpha-2) для привязки к координатам.

### 2. map.py (обработчик)
Скрипт на Python, который:
- Читает `data.json`
- Сопоставляет код страны с координатами (lat, lon) из встроенного словаря
- Для каждой точки создаёт CircleMarker на карте Folium
- Цвет маркера определяется языковой группой
- В popup выводится: слово, транскрипция, группа, справка
- Сохраняет карту в `sister_map.html`

### 3. sister_map.html (выход)
Автономный HTML-файл на Leaflet (JS). Открывается в браузере без сервера.

## Зависимости
- Python 3.8+
- Folium (Leaflet.js обёртка)
- Нет внешних API или БД
