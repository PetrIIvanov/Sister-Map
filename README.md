# SisterMap

Интерактивная карта Европы, показывающая слово **«сестра»** на языках разных стран.

## Цель
Наглядно увидеть, как одно и то же понятие звучит в родственных и неродственных языках, и как языковые группы распределены географически.

## Языковые группы

| Группа     | Цвет      | Пример          |
|------------|-----------|-----------------|
| Славянская | Красный   | сестра (RU)     |
| Германская | Синий     | sister (GB)     |
| Романская  | Зелёный   | sœur (FR)       |
| Уральская  | Фиолетовый| sisko (FI)      |
| Балтийская | Оранжевый | māsa (LV)       |
| Прочие     | Серый     | αδελφή (GR)     |

## Структура репозитория

```
├── CHANGELOG.md       # лог изменений
├── data.json          # исходные данные
├── map.py             # скрипт генерации карты
├── docs/              # документация
├── logs/              # файлы логов
├── requirements.txt   # зависимости
└── sister_map.html    # результат (генерируется)
```

## Быстрый старт

```bash
pip install -r requirements.txt
python map.py
```

Откройте `sister_map.html` в браузере.

## Логгирование

Скрипт `map.py` пишет логи двух уровней:
- **INFO** — основные шаги (загрузка, создание карты, сохранение)
- **ERROR** — ошибки (файл не найден, сбой генерации)

Логи выводятся в stdout и дописываются в `logs/sister_map.log`.

## Данные
Файл `data.json` содержит 33 записи с полями:
- `code` — ISO alpha-2 код страны
- `country_name` — название страны
- `word` — слово «сестра» на местном языке
- `transcription` — фонетическая транскрипция
- `ru_reading` — произношение кириллицей
- `group` — языковая группа
- `info` — историко-лингвистическая справка
